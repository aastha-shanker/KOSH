# 🚀 KOSH - Full Stack Web Application

A full-stack web application built as a learning + practical implementation project using **HTML, CSS, JavaScript (frontend)** and **Node.js, Express, MongoDB (backend)**.

---

## 🔥 Overview

KOSH is a full-stack web application that demonstrates user authentication, backend API development, and frontend-backend integration. The project focuses on building a real-world structure similar to production applications.

---

## 📁 Project Structure

```
Project - KOSH/
│
├── kosh-frontend/     # Frontend (UI layer)
├── kosh-backend/      # Backend (API + database logic)
├── node_modules/
└── .gitignore
```

---


## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript (Vanilla)

### Backend

* Node.js
* Express.js
* MongoDB + Mongoose
* JSON Web Token (JWT)
* Middleware-based architecture

---


## 🔗 Project Flow

1. User interacts with frontend UI
2. Frontend sends API requests to backend
3. Backend processes authentication and data
4. MongoDB stores and retrieves user data
5. Response sent back to frontend

---

## ⚠️ Important Notes

* `.env` file is not included for security reasons
* Backend must be running for full functionality
* Ensure MongoDB connection is active before testing APIs
